# 📊 每日 arXiv 论文推荐

**研究画像:**
- 领域: Computer Graphics; 3D Human Motion Generation
- 方法: 视频生成模型（万相、Seedance）, 视觉语言大模型（VLM）, 三维运动生成模型, 视觉引导特征提取
- 痛点: 从二维视频中提取可靠的三维运动先验信息的挑战, 如何有效筛选高质量视频生成结果的质量评估问题, 视觉引导特征与三维运动模型之间的特征对齐与融合难题

**最相关的 5 篇论文:**


## 📄 论文 1 - 相关性: 35%
**标题:** VideoWeave: Unlocking Geometric Consistency in Video Generation via Joint Geometry-Video Modeling
**作者:** Xunzhi Xiang, Zixuan Duan, Yabo Chen
**arXiv:** https://arxiv.org/abs/2606.14162

**相关性说明:** 该论文是关于通用视频生成的扩散模型方法，与研究_profile的3D人体运动生成应用领域不同。研究_profile需要的是从2D视频中提取人体运动的技术，而该论文关注的是视频生成中的几何一致性问题。虽然两者都使用扩散模型，但应用领域完全不同（视频生成 vs 人体运动生成）。该论文没有涉及人体运动的提取、先验信息学习或质量评估等痛点。

**摘要:** VideoWeave是一种潜空间后训练框架，通过使用隐式几何模型特征来约束生成分布，在共享去噪空间中联合建模几何潜变量和视频潜变量，从而在不需要显式几何重建的情况下改善视频生成的3D结构一致性。


## 📄 论文 2 - 相关性: 30%
**标题:** Temporal Backtracking Search for Test-time Generative Video Reasoning
**作者:** Sejoon Jun, Zheng Ding, Huangyuan Su
**arXiv:** https://arxiv.org/abs/2606.13861

**相关性说明:** 该论文是关于通用视频生成的测试时推理优化，与3D人体运动生成的应用领域完全不同。论文关注的是生成2D视频帧内容（在算法、导航、机器人领域测试），而非从2D视频中提取3D人体运动或生成3D运动序列。虽然都涉及视频生成方法，但研究目标完全不同：一个是生成视觉内容，另一个是生成三维人体运动骨架序列。论文未涉及视觉引导特征与三维运动模型的特征对齐与融合，也未解决从二维视频提取三维运动先验的核心问题。

**摘要:** 针对视频生成推理的测试时缩放问题，提出 Temporal Backtracking Search (TBS) 方法，通过在时间轴上搜索而非去噪步骤或根级重采样来提升视频模型的推理能力。


## 📄 论文 3 - 相关性: 30%
**标题:** Instruct-Particulate: Scaling Feed-Forward 3D Object Articulation with Kinematic Control
**作者:** Ruining Li, Yuxin Yao, Matt Zhou
**arXiv:** https://arxiv.org/abs/2606.14699

**相关性说明:** Application domain mismatch: Paper is about 3D object articulation (关节化3D对象) for kinematic control, while research profile is about 3D human motion generation (三维人体运动生成). These are fundamentally different - object articulation deals with mechanical joints of 3D objects (like drawers, doors), not human motion. The paper uses 3D mesh input and predicts articulation parameters, which is unrelated to video-based human motion generation or video generation models. Although both involve 3D understanding, the core application domain and methods are completely different.

**摘要:** Instruct-Particulate是一个能从3D网格预测关节化部件分割和运动参数的神经网络模型，通过引入目标运动学规范（包括部件描述、连接性、关节类型等）来解决训练数据稀缺问题。


## 📄 论文 4 - 相关性: 25%
**标题:** TSA: Temporal Slot Activation for Persistent Object-Centric Video Representation
**作者:** Duc Nguyen, Sieu Tran, Hao Vo
**arXiv:** https://arxiv.org/abs/2606.13714

**相关性说明:** This paper focuses on unsupervised video object-centric representation learning using slot attention mechanisms for object decomposition and tracking in videos. While it deals with video analysis, its application domain is fundamentally different from 3D Human Motion Generation. The paper addresses object segmentation and identity preservation in videos (MOVi-C/E, YT-VIS, OVIS benchmarks), not 3D human pose/motion extraction or generation. The research profile specifically needs methods for extracting 3D motion priors from 2D videos and generating 3D human motion, which this paper does not address. Using slot attention for video object decomposition is a different method in a different application domain (generic video object understanding vs human motion generation).

**摘要:** 本文提出时间槽激活(TSA)机制，解决无监督视频对象中心学习中持久槽在目标缺席或遮挡时的状态漂移和重建干扰问题。TSA学习每个槽每帧的激活分数，用于控制槽的状态更新和参与解码器，在目标不可见时保持槽的先前状态。


## 📄 论文 5 - 相关性: 25%
**标题:** CineOrchestra: Unified Entity-Centric Conditioning for Cinematic Video Generation
**作者:** Sharath Girish, Tsai-Shien Chen, Zhikang Dong
**arXiv:** https://arxiv.org/abs/2606.13768

**相关性说明:** 该论文是关于电影视频生成的扩散模型，与研究Profile中的三维人体运动生成应用领域完全不同。虽然都涉及视频生成，但研究Profile关注的是从二维视频中提取三维运动先验、生成可应用于角色的三维运动数据，而该论文关注的是生成电影级二维视频内容（包括多主体、摄像机控制等）。两者虽都用扩散模型，但应用领域截然不同：一个是三维运动生成，一个是二维视频生成。

**摘要:** CineOrchestra是一个统一的视频扩散模型，能够同时控制多主体、事件、摄像机和镜头切换，通过实体中心调节原语和协调旋转嵌入实现精细化电影级视频生成。
