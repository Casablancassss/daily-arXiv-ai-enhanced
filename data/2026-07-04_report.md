## 📄 论文 1 - 相关性: 55%
**标题:** Alignment Is All You Need For X-to-4D Generation
**作者:** Qiaowei Miao, Kehan Li, Yawei Luo
**arXiv:** https://arxiv.org/abs/2607.02516

**相关性说明:** The paper is in Computer Graphics and deals with 3D motion generation from visual inputs (video), which is methodologically similar to the research profile. However, there's a significant domain mismatch: the research profile focuses on human motion synthesis from 2D videos (addressing human motion generation with methods like VLM quality supervision), while this paper focuses on general 4D object/scene generation (dynamic 3D objects/scenes) rather than human motion. The paper uses alignment techniques for X-to-4D which differs from the specific human motion synthesis pipeline mentioned in the profile.

**摘要:** Align4D是一个灵活的框架，可以将任意模态输入（文本、图像、视频、3D）转换为连贯的4D内容（视频-3D对）。该方法通过视频引导4D运动，通过3D数据塑造4D几何形状，实现了任意用户定义的X-to-4D生成。


## 📄 论文 2 - 相关性: 50%
**标题:** InterCMDM: Block-Causal Diffusion for Autoregressive Human Interaction Generation
**作者:** Qing Yu, Kent Fujiwara
**arXiv:** https://arxiv.org/abs/2607.01743

**相关性说明:** While both papers focus on 3D human motion generation (same application domain), this paper addresses a DIFFERENT problem. The research profile focuses on extracting motion from 2D VISUAL inputs (videos) and using video generation models, whereas InterCMDM focuses on text-to-motion generation with diffusion models. The pain points in the profile (2D-3D domain gap, video-to-motion extraction, VLM quality supervision) are NOT addressed. This paper uses text prompts as input, not visual features from videos. It's relevant to human motion generation but not to the specific two-stage video-to-motion synthesis pipeline described in the profile.

**摘要:** InterCMDM是一个用于双人交互生成的块因果潜在扩散框架，通过双流因果扩散变换器解决现有方法中因果关系模糊和时间漂移的问题，支持多种协调行为的可控交互生成。


## 📄 论文 3 - 相关性: 45%
**标题:** JointHOI: Jointly Generating Contact Maps Enhances Hand Object Interaction Generation
**作者:** Mingyeong Song, Jungbin Cho, Jisoo Kim
**arXiv:** https://arxiv.org/abs/2607.01768

**相关性说明:** While this paper is in Computer Graphics and addresses 3D motion generation, it uses a fundamentally different approach than the research profile. The profile focuses on two-stage pipelines using video generation models (Wan, Seedance) to extract visual features from 2D videos for 3D motion synthesis, whereas this paper uses a single-stage text-to-motion diffusion model without any video input. The pain points mentioned (domain gap between 2D visual features and 3D motion, video-based quality supervision, visual feature extraction) are not addressed. This paper generates hand-object interactions directly from text, which is a different paradigm than the video-to-3D motion pipeline specified in the research profile.

**摘要:** JointHOI是一个单阶段扩散框架，能够根据文本提示共同生成3D手部-物体运动和动态接触图，解决文本驱动的手-物体交互(HOI)生成中的物理真实性问题。


## 📄 论文 4 - 相关性: 45%
**标题:** Training-free Controllable Human Motion Generation under Heterogeneous Constraints
**作者:** Xiaofei Hui, Bo Yan, Haoxuan Qu
**arXiv:** https://arxiv.org/abs/2607.01990

**相关性说明:** While this paper is about human motion generation in computer graphics (matching the general field), it does not address the specific research profile. The profile focuses on a two-stage pipeline using video generation models (Wan, Seedance) as priors to extract 2D visual features for 3D motion synthesis. This paper instead focuses on training-free controllable motion generation under heterogeneous constraints using stochastic control. Key pain points from the profile (domain gap between 2D visual features and 3D motion, video-to-3D pipeline, VLM quality supervision) are not addressed. The paper uses diffusion for motion generation but lacks the video feature extraction and two-stage pipeline approach central to the research profile.

**摘要:** 提出MIC框架，首个无需训练的运动生成方法，可同时处理连续目标约束和基于准则的约束，将扩散模型视为随机控制问题


## 📄 论文 5 - 相关性: 45%

**作者:** Lu Pan, Hongwei Zhao
**arXiv:** https://arxiv.org/abs/2607.02034

**相关性说明:** The paper is in the Computer Graphics domain (3D human motion) but addresses a different application scenario. It focuses on Human-Scene Interaction (HSI) imitation from MoCap data, while the research profile specifically targets 2D video-to-3D motion generation using video generation models (Wan, Seedance), VLM supervision, and visual feature extraction. This is moderately relevant as it deals with 3D human motion but misses the key methodological requirements of the profile.

**摘要:** 本文提出ComplexMimic框架，解决复杂环境下人体-场景交互(HSI)的模仿学习问题，通过双流策略和难度感知蒸馏实现对不完美MoCap数据的物理 plausible 重建。
