## 📄 论文 1 - 相关性: 70%
**标题:** SiGnature: Explicit Motion Diffusion for Stylized Semantic Gesture
**作者:** Adi Rosenthal, Tomer Koren, Nadav Shaked
**arXiv:** https://arxiv.org/abs/2606.15889

**相关性说明:** The paper is about diffusion-based 3D human motion generation (co-speech gestures), which matches the core application domain (Computer Graphics/Vision for motion synthesis). However, it doesn't address the specific pain points in the profile: bridging 2D video features to 3D motion, VLM-based quality verification, or text-to-motion alignment. The paper focuses on semantic control and style preservation in gesture generation using a different approach (joint rotation space), not the 2D-to-3D domain gap or VLM verification mentioned in the profile. It's highly relevant for motion generation methods but misses the key technological innovations (VLM, multi-modal fusion) specified.

**摘要:** SiGnature是一个框架，用于在共语言手势生成中实现语义控制和风格保留的平衡。通过在显式联合旋转空间中操作，提出无需训练的联合运动集成(JMI)机制，可将外部语义手势注入扩散生成过程。


## 📄 论文 2 - 相关性: 65%
**标题:** MotionVLA: Vision-Language-Action Model for Humanoid Motion
**作者:** Nonghai Zhang, Siyu Zhai, Yanjun Li
**arXiv:** https://arxiv.org/abs/2606.15142

**相关性说明:** This paper directly addresses the core application domain of 3D human/humanoid motion generation from visual and textual inputs, which matches the research profile's focus on bridging 2D video features to 3D motion generation. However, there is a significant methodological mismatch: the paper uses a Vision-Language-Action (VLA) model rather than the Diffusion Models (Wanxiang, Seedance) specified in the methods. While the paper does use VLM (Qwen3.5) for visual feature extraction, it does not address quality control/verification of generated motions via VLM, which is one of the key pain points. The dual-stream frequency tokenization approach (DSFT) for temporal consistency is relevant but implemented differently from diffusion-based methods. Overall, highly relevant in application domain but methodologically different from what the profile specifically seeks.

**摘要:** 本文提出DSFT双流频率分词器和MotionVLA模型，解决单一码本无法同时处理低频姿态语义和高频物理动力学的问题，实现从场景图像和文本生成逼真人形运动。


## 📄 论文 3 - 相关性: 45%
**标题:** FUSE: Quantifying Uncertainty in Vision-Language Models by Bayesian Fusing Epistemic and Aleatoric Uncertainty
**作者:** Harry Zhang, Luca Carlone
**arXiv:** https://arxiv.org/abs/2606.14728

**相关性说明:** This paper addresses uncertainty quantification in Vision-Language Models using Bayesian methods. While it tangentially relates to the "Quality control and reliability verification" pain point in the research profile (since both involve VLMs), the application domain is fundamentally different - this paper focuses on general VLM uncertainty for robotics applications, not 3D motion generation in Computer Graphics. The paper doesn't address core challenges like bridging 2D video to 3D motion, temporal consistency, or text-to-motion alignment. It uses VLMs but for uncertainty quantification rather than motion synthesis, making it only moderately relevant despite sharing some methodological components.

**摘要:** FUSE是一个用于视觉语言模型不确定性量化的概率框架，通过贝叶斯融合机制结合两种不确定性来源（数据固有不确定性和模型认知不确定性），实现SOTA不确定性校准。


## 📄 论文 4 - 相关性: 45%
**标题:** Multi-HMR 2: Multi-Person Camera-Centric Human Detection, Mesh Recovery and Tracking
**作者:** Guénolé Fiche, Philippe Weinzaepfel, Romain Brégier
**arXiv:** https://arxiv.org/abs/2606.14841

**相关性说明:** The paper deals with human-related 3D geometry (human mesh recovery) in computer vision, but addresses a different core task than the research profile. The profile focuses on 3D MOTION GENERATION (creating new motions from scratch using diffusion models and text-to-motion alignment), while this paper focuses on ESTIMATING/recovering existing human poses from images. While both involve human bodies in 3D, motion generation vs mesh recovery are different application domains. The paper uses DETR-based methods rather than diffusion models for motion generation, and does not address text-to-motion alignment, VLM-based quality control, or multi-modal feature fusion for generation. The only tangential relevance is that both deal with 3D human representation, but the core problem and methods are substantially different.

**摘要:** Multi-HMR 2是一种基于DETR的多人camera-centric人体检测、网格恢复和跟踪框架，能够在没有真实相机内参的情况下预测场景一致的相机和人体网格，实现度量3D定位，并通过SAM2蒸馏的图像记忆特征实现跟踪。


## 📄 论文 5 - 相关性: 45%
**标题:** EmoZone-Talker: Regional Semantic Control of Audio-Driven 3DGS Talking Heads via Facial Action Units
**作者:** Tingting Chen, Shaojun Wang, Huaye Zhang
**arXiv:** https://arxiv.org/abs/2606.15848

**相关性说明:** This paper addresses facial animation/talking heads using 3D Gaussian Splatting, which is a different application domain from the research profile's focus on 3D motion generation (typically body motion like gestures). While it partially addresses temporal consistency (a pain point), it does not use diffusion models or VLM for quality control. The methods (SZ-PAB, CIT-AE) are specialized for facial Action Units rather than general 3D motion synthesis. The domain gap between facial animation and general 3D motion generation is significant - this is moderately relevant within Computer Graphics but doesn't match the specific methods and application focus.

**摘要:** EmoZone-Talker是一种基于3D Gaussian Splatting的面部动画框架，通过创新性地引入解剖学先验引导的空间解耦(SZ-PAB)和时序一致的表情动作编码器(CIT-AE)，实现了对说话人头合成中面部表情的精细化、可解释化和可编辑化控制。
